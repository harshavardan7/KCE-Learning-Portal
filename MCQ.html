Java Basic Syntax

Java's basic syntax is the foundation for writing Java programs. Understanding these fundamentals is crucial for any Java developer.

1. Java Program Structure

Every Java program starts with a class definition. The class name should match the filename.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2. Main Method

The main method is the entry point of any Java application:

public static void main(String[] args) {
    // Your code here
}

3. Statements and Semicolons

In Java, each statement must end with a semicolon (;).

int x = 5;
System.out.println("The value of x is: " + x);

4. Comments

Java supports single-line and multi-line comments:

// This is a single-line comment

/* This is a
   multi-line comment */

5. Case Sensitivity

Java is case-sensitive. 'HelloWorld' and 'helloworld' are considered different identifiers.

Note: Consistent naming conventions are important for code readability. Class names typically start with an uppercase letter, while method and variable names start with a lowercase letter.

6. Blocks of Code

Java uses curly braces {} to define blocks of code:

if (condition) {
    // code block
} else {
    // another code block
}
